# Neplog

这是一个基于Springboot + Hibernate + Redis + Vue开发的轻量级前后端分离个人博客项目的后端部分。
同时也是一个个人以学习为主要目的的项目，主要是对以上所有技术的使用层面上的熟悉。

在技术上都有向[Halo]()和[eladmin]()学习

前端仓库见[neplog-web](https://github.com/Uzemiu/neplog-web)

目前项目正在开发中...
